Elementary Cellular Automata

This is a very simple implementation of Wolfram's Elementary Automata. Simply plug in the values you would like to experiment with, and either click generate, or press enter.

  • Rule: Determines the neighbourhood update rules
  • Width: The number of cells that will be generated horizontally
  • Number of iterations: The number of iterations that will be stacked on one another (i.e. vertical height)
  • Generation delay: The buffer between the generation of each iteration - a value of 0 means instant generation
